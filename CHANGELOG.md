## [1.2.0] - 2025-12-12

### Ajouté
- Module `src/signature.py` : calcul des signatures angulaires
- Module `src/palimpseste.py` : mélange des grilles Guasti/Pythagore
- Module `src/__init__.py` : initialisation du package Python
- Documentation `docs/fiche_memo.md` : résumé d'une page
- Documentation `docs/contribution_theorie_nombres.md` : analyse des apports
- Fichier `requirements.txt` : dépendances Python
- Fichier `LICENSE` : licence MIT

### Documenté
- Critère du 45° pour les carrés parfaits
- Convergence des premiers jumeaux vers 45°
- Hiérarchie des puissances (n¹, n², n³)
- Palimpsestes arithmétiques (additif, multiplicatif, PGCD)
- Théorème : PGCD(G, P) = G (Guasti est le noyau de Pythagore)
